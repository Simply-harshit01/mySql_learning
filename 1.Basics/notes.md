# 📘 Day 1 – DBMS Basics

## 🔹 What is a Database?

A database is an organized collection of data that can be easily accessed, managed, and updated.

Example:

* Student records
* Library system
* Banking data

---

## 🔹 DBMS (Database Management System)

DBMS is software used to manage databases.

Functions:

* Store data
* Retrieve data
* Control access
* Ensure security

Examples:

* MySQL
* Oracle
* PostgreSQL

---

## 🔹 RDBMS (Relational DBMS)

RDBMS stores data in the form of **tables (rows and columns)**.

Key Features:

* Uses Primary Key
* Maintains relationships
* Reduces redundancy

MySQL is an RDBMS.



## 🔹 SQL (Structured Query Language)

SQL is used to communicate with the database.

Used For:

* Creating tables
* Inserting data
* Retrieving data



## 🔹 Types of SQL Commands

| Type | Full Form                  | Example       |
| ---- | -------------------------- | ------------- |
| DDL  | Data Definition Language   | CREATE, ALTER |
| DML  | Data Manipulation Language | INSERT        |
| DQL  | Data Query Language        | SELECT        |
| DCL  | Data Control Language      | GRANT         |
| TCL  | Transaction Control        | COMMIT        |



## 🔹 Data Types Learned

* `INT` → Stores numbers
* `VARCHAR(n)` → Stores text
* `AUTO_INCREMENT` → Automatically increases value



## 🔹 Commands Practiced Today


CREATE TABLE Students (...);
INSERT INTO Students (...);
SELECT * FROM Students;
